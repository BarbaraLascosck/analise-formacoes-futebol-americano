<h1 align="center"> Detecção Automatizada para Análise de Formações Defensivas de Futebol Americano com Visão Computacional </h1>

### Este projeto apresenta o desenvolvimento de um sistema de Visão Computacional treinado com técnicas de Inteligência Artificial (IA) para identificar a posição e a quantidade de jogadores específicos em campo, com o objetivo de auxiliar na identificação de formações defensivas no futebol americano.

### Foram utilizadas imagens do pré-snap (antes da jogada acontecer) rotuladas e treinadas na plataforma Roboflow, integradas a um script em Python para realizar a detecção automática. O sistema reconhece jogadores e a formação defensiva e exibe o resultado ao usuário, contribuindo para uma leitura mais rápida da defesa adversária durante a análise de jogadas.

### Tecnologias Utilizadas: Python, Visão Computacional (Roboflow), Inteligência Artificial (IA).

<h1 align="center"> Automated Detection of Football Defensive Formations Using Computer Vision </h1>

### This project presents the development of a Computer Vision system trained with Artificial Intelligence (AI) techniques to identify the position and number of specific players on the field, aiming to assist in the detection of defensive formations in Football. 

### Labeled images were trained using the Roboflow platform and integrated into a Python script for automatic detection. The system recognizes the defensive formation and displays the result to the user, contributing to a faster and more efficient tactical analysis during play reviews.

### Tecnologies: Python, Computer Vision (Roboflow); Artificial Intelligence (AI).
